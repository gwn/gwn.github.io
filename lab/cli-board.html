#Project Bootstrapping

Whenever we start a new project together, you will be provided a link to the project's wiki and some instructions (if any) by the project manager as soon as the project is bootstrapped. The project will be assigned an owner from both your and our sides. Then things will proceed according to the project lifecycle.
